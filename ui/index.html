Stop Supply Chain Attacks Cold.

What is the point of this? When you run pip install or npm install, packages can run arbitrary code to steal your SSH keys or `.env` files and send them to an attacker's server.

dependency-jail intercepts system calls via LD_PRELOAD to block all unauthorized outbound network connections during installation, securing your environment instantly.

Why it's necessary.

Zero-Root Sandboxing

No Docker, no VMs, no root privileges required. Works by injecting a custom libc.so connect() hook that drops malicious sockets at the kernel boundary.

Profile-Based Trust

Use the --profile pypi flag to automatically trust only Python registries, immediately isolating packages that try to reach out to npm, github, or arbitrary IPs.

Rich JSON Reporting

Use --report output.json to generate highly detailed forensic logs of everything the installer attempted to do. View them visually in our local Report Dashboard.
